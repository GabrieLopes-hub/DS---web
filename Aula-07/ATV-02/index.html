controls: Exibe controles padrão, como play, pause, volume, etc.



autoplay: Inicia o vídeo automaticamente quando a página é carregada
(usar com muted para evitar reprodução de som sem aviso).



loop: Faz com que o vídeo reinicie automaticamente após terminar, útil para vídeos de fundo ou clipes curtos.



muted: Silencia o áudio do vídeo (necessário para vídeos que iniciam automaticamente).



poster: Define uma imagem que aparece enquanto o vídeo carrega ou antes de ser reproduzido.
